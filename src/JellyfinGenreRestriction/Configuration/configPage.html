Ultimate Parental Control Plugin

Configure automated item tagging. Once items are tagged, use Jellyfin's built-in Parental Control settings per-user to block those tags.

1. Genre Blacklist

Automatically add tags to items matching these genres.

2. Studio Blacklist

Automatically add tags to items produced by these studios.

3. Keyword Deep Scan Blacklist

Scans the item's Title and Overview/Description for forbidden words (case-insensitive) and adds a tag.

4. Whitelist Mode

If enabled, any item that does NOT match your Safe Genres, Safe Studios, or Safe Keywords will automatically receive the Fallback Restricted Tag.
