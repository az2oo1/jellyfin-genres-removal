Genre Restriction Plugin

Use this page to map Genres to Tags. When the scheduled task runs, matching genres will automatically add the corresponding tags to your media items. You can then use Jellyfin's built-in Parental Controls to block those specific tags for any user.

Example:
{
  "Horror": "RESTRICTED_HORROR",
  "Adult": "RESTRICTED_ADULT"
}
Turn on the scheduled task for synchronizing tags.
